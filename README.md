# customer-churn-analysis

## Project Overview 
This project analysis customer churn data to identify factors affecting customer retention.

## Tools Used
- Excel
- pandas
- Python (Pandas, Matplotlib)
- numpy

## Analysis perfromed
- Data cleaning
- Missing value Analysis
- Churn percentage calculation
- Gender-wise churn Analysis
- Monthly charges Analysis

## Conclusion
Customers with higher monthly charges showed a higher tendency to churn.
